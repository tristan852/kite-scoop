# Scoop Bucket for Kite

This is a Scoop bucket for installing [Kite](https://github.com/tristan852/kite), a lightweight Connect Four solver CLI.

## Installation

First, add this bucket:

```powershell
scoop bucket add kite https://github.com/tristan852/kite-scoop
````

Then install Kite:

```powershell
scoop install kite
```

## Updating

To update to the latest version:

```powershell
scoop update kite
```

## About Kite

Kite is a fast command-line Connect Four solver that can:

* Analyze board positions
* Compute optimal moves
* Play against bots of varying skill levels
* Provide interactive gameplay
